# Сводная AD / MFA / Кадры

Веб-приложение для загрузки выгрузок из Active Directory, MFA и кадровой системы (Develonica.People) и просмотра сводной таблицы по сотрудникам с сверкой данных: email, телефон, ФИО между системами.

Запуск на сервере (Ubuntu): из корня проекта выполнить `./install.sh`. Подробные технические инструкции по запуску хранятся в отдельном файле в проекте и не публикуются в репозитории.
